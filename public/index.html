Querying the API

Substitute all example URLs with the base https://marsphotoapi.onrender.com/. The API can be queried by Martian sol, by Earth date, or filtered by camera.

Queries by Martian sol

https://marsphotoapi.onrender.com/api/v1/rovers/curiosity/photos?sol=1000

Queries by Earth date

https://marsphotoapi.onrender.com/api/v1/rovers/curiosity/photos?earth_date=2015-06-03

Filter by camera

https://marsphotoapi.onrender.com/api/v1/rovers/curiosity/photos?sol=1000&camera=fhaz

Paginated results

Results are limited to 25 photos per call; additional pages can be retrieved via the page parameter:

https://marsphotoapi.onrender.com/api/v1/rovers/curiosity/photos?sol=1000&page=2

The database will be updated regularly with the latest photos. For details of available cameras and mission manifests, consult the README on GitHub.
